# Collaborative Grocery Lists

> **Phase:** 4 — Social and Household Utility  
> **Target release:** Family/shared beta (February 1, 2027)  
> **Milestone:** Year 1 expansion release (March 19, 2027)  
> **Tier:** Family plan ($79.99/year)  
> **Priority:** High

---

## Summary

Enable household members to share and collaboratively edit grocery lists in real time. When the family meal plan is set for the week, generate a shared grocery list that everyone can view, check off items from, and add to.

---

## Value Proposition

- **For users:** One shared grocery list for the household. Anyone can check off items while shopping. Add items from any device. No more duplicate purchases or "I thought you were getting the milk" miscommunications.
- **For the business:** Collaborative grocery lists are one of the most requested features in household cooking apps. They lock in the Family plan value and increase daily engagement across multiple users.
- **Habit formation:** A shared grocery list is used multiple times per week — more frequent touchpoints mean stronger retention.

---

## Detailed Instructions

### 1. Shared list creation

- Generate a grocery list from the shared household meal plan (integrates with meal planner and grocery list features).
- Household members can also create standalone shared grocery lists (not tied to a meal plan).

### 2. Real-time collaboration

- All household members see the same list and can:
  - Check off items.
  - Add new items.
  - Remove items.
  - Edit item quantities or notes.
- Changes sync in real time across all members' devices.
- Show who last modified an item (optional).

### 3. Notifications

- Optional push notification when a household member adds items to the shared list.
- Optional reminder notification before a scheduled grocery trip (if integrated with calendar, otherwise skip).

### 4. List management

- Multiple shared lists can exist simultaneously (e.g., "Weekly groceries," "Costco run," "Party supplies").
- Archive completed lists for reference.
- Clear all checked items with one action.

### 5. Individual items

- Items can be added manually (not just from recipes).
- Support categorization: produce, dairy, protein, pantry, frozen, household, etc.
- Support notes on items (e.g., "organic only," "the brand Mom likes").
